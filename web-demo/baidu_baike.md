# 南半球历法

**南半球历法**，是基于太极对称原理，将传统北半球中国历法系统转换为南半球对应体系的一套历法算法。该历法由南半球万年历（Nanbanqiu Wannianli）项目提出，自成一派，旨在为南半球华人社区提供符合当地天文现象与季节规律的农历参照体系。

> ⚠️ **重要声明**：南半球历法为独立历法体系，其属性、推演逻辑与北半球传统术数体系存在根本差异。本历法**不适用于**北半球术数推演、择吉、风水堪舆等传统应用，**严禁用于**任何商业决策、财务投资、医疗健康等实际事务。详见文末法律声明。

## 一、产生背景

传统中国历法（农历）拥有四千余年历史，其天文观测基础完全建立在北半球中纬度地区。农历的月份划分以朔望月为基准，节气以太阳黄经为依据，四柱八字以当地时辰为计算起点——这一切均以北半球的季节循环为默认前提。

然而，全球超过3000万华人居住在南半球（阿根廷、澳大利亚、新西兰、巴西、南非等），当北半球庆祝春节（农历正月初一）时，南半球正值盛夏；当北半球立春万物复苏，南半球立秋万物萧瑟。传统历法直接应用于南半球，存在根本性的季节错位。

南半球历法正是为解决这一错位而诞生。它不是对传统历法的否定，而是基于太极阴阳哲学的补全——正如太极包含阴阳，完整的中国历法体系应当涵盖南北两个半球。

## 二、核心算法

南半球历法的所有换算均基于一条核心原理：**太极180°旋转对称**。南北半球在天文、地理、气象上呈对跖关系，历法换算即是对这种对跖关系的数学表达。

### 2.1 阳历换算

| 项目 | 公式 | 说明 |
|------|------|------|
| 月份 | 南半球月份 = 北半球月份 + 6（超过12则减12） | 阳历月份直接偏移半年 |
| 年份 | 北半球月份 < 7 → 南半球年份 = 北半球年份 - 1；北半球月份 ≥ 7 → 南半球年份 = 北半球年份 | 跨年边界处理 |

**示例**：
- 北半球2026年1月 → 南半球2025年7月
- 北半球2026年7月 → 南半球2026年1月

### 2.2 农历换算

| 项目 | 公式 | 说明 |
|------|------|------|
| 农历月份 | 南半球农历月 = 北半球农历月 + 6（超过12则减12） | 农历月份同步偏移 |
| 闰月 | 南半球闰月 = (北半球闰月 + 6 - 1) % 12 + 1 | 闰月同步偏移 |
| 年份 | 同阳历跨年规则 | — |

### 2.3 太极对立日

**公式**：南半球农历日 = 北半球农历日 + 15（超过当月天数则减当月天数，月份进位）

**本质**：农历日+15意味着在朔望月周期上相隔180°，月相形态恰好翻转。这正是太极双鱼图中两条鱼眼的对望关系。

**示例**：北半球农历四月十八 → 南半球农历十月初三

### 2.4 节气对应

**公式**：北半球第N个节气 ↔ 南半球第(N+12)个节气（超过24则减24）

完整对应表：

| 北半球 | 南半球 | 北半球 | 南半球 |
|--------|--------|--------|--------|
| 立春 | 立秋 | 立夏 | 立冬 |
| 雨水 | 处暑 | 小满 | 小雪 |
| 惊蛰 | 白露 | 芒种 | 大雪 |
| 春分 | 秋分 | 夏至 | 冬至 |
| 清明 | 寒露 | 小暑 | 大寒 |
| 谷雨 | 霜降 | 大暑 | 小寒 |

### 2.5 月相镜像

南北半球月相亮面朝向呈镜像对称：

| 北半球 | 南半球 |
|--------|--------|
| D形蛾眉月（右侧渐亮） | 反D形蛾眉月（左侧渐亮） |
| 上弦月（右半亮） | 上弦月（左半亮） |
| ☾ 月相符号 | ☽ 月相符号（旋转180°） |

### 2.6 四柱八字对称

**公式**：南半球四柱 = 北半球四柱各偏移+6
- 天干：南半球天干 = (北半球天干序号 + 6) % 10
- 地支：南半球地支 = (北半球地支序号 + 6) % 12

**示例**：北半球丙午 → 南半球壬子

## 三、南北半球属性颠倒

南半球历法的根本逻辑基础，在于南北半球在地理、气候、天文上的**属性颠倒**。这种颠倒不是人为设定，而是客观物理事实。

### 3.1 温度方向颠倒

| 属性 | 北半球 | 南半球 |
|------|--------|--------|
| 往北 | 越冷（远离赤道，靠近北极） | 越热（靠近赤道） |
| 往南 | 越热（靠近赤道） | 越冷（远离赤道，靠近南极） |
| 极端寒冷区 | 北极 | 南极 |
| 热带区域 | 赤道以北 | 赤道以南 |

**关键结论**：北半球的"北"属性为寒，南半球的"北"属性为热。同是"北方"，南北半球含义完全相反。

### 3.2 季节方向颠倒

| 时间 | 北半球 | 南半球 |
|------|--------|--------|
| 1月 | 冬季（寒冷） | 夏季（炎热） |
| 4月 | 春季（升温） | 秋季（降温） |
| 7月 | 夏季（炎热） | 冬季（寒冷） |
| 10月 | 秋季（降温） | 春季（升温） |

### 3.3 太阳运行方向颠倒

| 属性 | 北半球 | 南半球 |
|------|--------|--------|
| 太阳升起的方位 | 东方偏南 | 东方偏北 |
| 正午太阳方位 | 南方天空 | 北方天空 |
| 太阳落下的方位 | 西方偏南 | 西方偏北 |
| 阳面（向阳面） | 朝南 | 朝北 |

### 3.4 旋转方向颠倒

| 系统 | 北半球 | 南半球 |
|------|--------|--------|
| 台风/飓风/气旋 | 逆时针 ↺ | 顺时针 ↻ |
| 反气旋（高压） | 顺时针 ↻ | 逆时针 ↺ |
| 大洋环流（副热带） | 顺时针 ↻ | 逆时针 ↺ |
| 地球自转方向 | 逆时针 | 顺时针 |

### 3.5 星象颠倒

| 属性 | 北半球 | 南半球 |
|------|--------|--------|
| 北极星 | 可见（指向正北） | 不可见（地平线以下） |
| 北斗七星 | 可见 | 不可见 |
| 南十字座 | 不可见 | 可见（指向正南） |
| 半人马座 | 低空可见 | 高空可见 |
| 银河中心方向 | 南方天空 | 北方天空 |

### 3.6 五行方位属性颠倒

传统五行方位理论基于北半球观测：

| 方位 | 北半球五行 | 南半球实际属性 |
|------|-----------|---------------|
| 北 | 水（寒） | 火（热，靠近赤道） |
| 南 | 火（热） | 水（寒，靠近南极） |
| 东 | 木 | 木（日出方位，属性不变） |
| 西 | 金 | 金（日落方位，属性不变） |
| 中 | 土 | 土（属性不变） |

**核心结论**：南北半球的"南北"五行属性完全颠倒，东西属性不变。这意味着直接将北半球五行方位理论套用于南半球，在寒热属性上会产生根本性错误。

## 四、月相验证

月相是南半球历法最直观的天然验证工具。

### 4.1 全球统一，朝向对立

- 月相由月球绕地球公转产生，全球同一时刻月相相同
- 但南北半球看到的亮面朝向呈镜像对称
- 北半球：D形蛾眉月（右侧渐亮）
- 南半球：反D形蛾眉月（左侧渐亮）

### 4.2 太极对立日的验证

1. 观测当前月相 → 确定农历日（全球一致）
2. 北半球农历日 + 15 = 南半球太极对立日
3. 例：北半球初八（上弦月，右半亮）↔ 南半球廿三（下弦月，左半亮）——镜像对称
4. 这为+15日偏移提供了实证验证

### 4.3 统一与对立

- **统一**：月相日数全球一致，满月全球同圆
- **对立**：亮面朝向镜像对称，渐圆方向相反
- **太极**：朔望圆上相隔180°，正是太极双鱼图的旋转对称

## 六、四柱八字对称

### 6.1 时辰与当地时间

四柱八字以当地时辰为计算基准。同一时刻，不同时区可能属于不同时辰，因此四柱必然不同。

### 6.2 南北半球四柱差异

| 柱别 | 差异原因 | 是否必然不同 |
|------|---------|-------------|
| 年柱 | 以立春为界，南北半球立春相差6个月 | 可能不同 |
| 月柱 | 以节气为界，节气对称相反 | 地支必然不同 |
| 日柱 | 以子时为界，时差可能导致跨日 | 可能不同 |
| 时柱 | 直接取决于当地时辰 | 必然不同 |

### 6.3 太极对称四柱

南半球四柱 = 北半球四柱各偏移+6：
- 天干+6%10：甲→庚，乙→辛，丙→壬，丁→癸，戊→甲，己→乙，庚→丙，辛→丁，壬→戊，癸→己
- 地支+6%12：子→午，丑→未，寅→申，卯→酉，辰→戌，巳→亥，午→子，未→丑，申→寅，酉→卯，戌→辰，亥→巳

## 七、星象定节气

南半球无法使用北斗七星定节气，替代方案如下：

### 7.1 半人马座指针法

α Centauri（南门二）与β Centauri（马腹一）连线延伸约4.5倍，指向南天极。功能类似北半球北斗七星。

### 7.2 天蝎座心宿二法

Antares（心宿二，视星等1.0）南北半球均可见。北半球心宿二黄昏高悬=夏季，南半球同时心宿二黄昏高悬=冬季。季节含义恰好相反——太极对称。

### 7.3 老人星法与麦哲伦星云法

Canopus（老人星，视星等-0.74）高度角随季节变化；大小麦哲伦星云绕南天极旋转，方位角指示季节。

## 八、科里奥利力验证

科里奥利力是地球自转产生的惯性力，数学表达式：

**F = -2m(ω × v)**

- 北半球：运动物体向右偏转
- 南半球：运动物体向左偏转

这从物理学上证明了南北半球旋转属性的颠倒，为历法属性颠倒提供了力学基础。

## 九、与北半球术数体系的根本区别

### 9.1 自成一派

南半球历法是一套独立的历法体系，其算法逻辑、属性推演均基于南半球天文地理事实，与北半球传统术数体系存在根本差异。

### 9.2 核心差异

| 项目 | 北半球传统术数 | 南半球历法 |
|------|--------------|-----------|
| 方位寒热 | 北寒南热 | 北热南寒 |
| 五行方位 | 北水南火 | 北火南水 |
| 季节对应 | 正月=春季 | 正月=秋季 |
| 旋转方向 | 逆时针为正 | 顺时针为正 |
| 太阳方位 | 正午在南方 | 正午在北方 |
| 阳面朝向 | 朝南 | 朝北 |

### 9.3 不兼容声明

南半球历法的属性推演结果**不可**直接套用北半球术数体系进行解读。原因：

1. **方位属性颠倒**：北半球的"北方"属寒，南半球的"北方"属热，同一方位含义完全不同
2. **季节循环反转**：北半球春生夏长，南半球同时秋收冬藏，五行生克关系需重新审视
3. **旋转方向相反**：气旋、环流方向相反，风水堪舆中的"气"的运行方向亦相反
4. **星象体系不同**：北极星不可见，北斗七星不可见，传统星象术数基础不存在

**因此，南半球历法自成一派，其推演结果不适用于北半球术数体系，北半球术数体系亦不适用于南半球。**
