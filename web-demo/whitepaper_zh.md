## 二、核心换算原理

### 2.1 阳历换算

南半球阳历月份与北半球的对应关系为：

$$M_{南} = M_{北} + 6 \quad (\text{若超过12则减12})$$

年份边界规则：

- 若北半球月份 < 7，则南半球年份 = 北半球年份 - 1
- 若北半球月份 ≥ 7，则南半球年份 = 北半球年份

**示例：**

| 北半球 | 南半球 |
|--------|--------|
| 2026年1月 | 2025年7月 |
| 2026年2月 | 2025年8月 |
| 2026年6月 | 2025年12月 |
| 2026年7月 | 2026年1月 |
| 2026年12月 | 2026年6月 |

### 2.2 农历换算

南半球农历月份与北半球的对应关系为：

$$M_{南}^{农历} = M_{北}^{农历} + 6 \quad (\text{若超过12则减12})$$

闰月换算规则：

$$\text{闰月}_{南} = (\text{闰月}_{北} + 6 - 1) \bmod 12 + 1$$

年份边界与阳历相同：北半球农历月份 < 7 时，南半球农历年份减一；≥ 7 时，年份不变。

### 2.3 太极对立日

南半球农历日期与北半球的对应关系为：

$$D_{南} = D_{北} + 15 \quad (\text{若超过当月天数则减去当月天数，进位至下月})$$

**本质：** 在朔望周期（约29.53天）上，加15天恰好对应180°的角距离，即月相完全反转——北半球的朔对应南半球的望，北半球的上弦月对应南半球的下弦月。这正是太极图中双鱼180°旋转对称的历法体现。

**示例：**

北半球农历四月十八 → 南半球农历十月初三（18 + 15 = 33，减去当月29天，余4日，进位至下月，即十月初三，具体日期取决于当月天数）。

## 三、月相验证

### 3.1 月相镜像对称

月球相位在全球同一时刻是相同的——这是天文学的基本事实。然而，南北半球观测者所见的月球被照亮一侧呈镜像对称：

- **北半球：** 盈月（上弦前）呈"D"形，右侧先亮；上弦月右半明亮
- **南半球：** 盈月（上弦前）呈反"D"形，左侧先亮；上弦月左半明亮
- **满月：** 南北半球所见完全相同，皆为圆满之月
- **亏月方向：** 北半球自右向左亏损，南半球自左向右亏损——盈亏方向恰好相反

这种镜像关系源于南北半球观测者相对于黄道面的上下位置颠倒，恰如太极图中双鱼的对称凝视。

### 3.2 验证方法

太极对立日的 +15 日偏移可通过以下步骤进行实证验证：

1. **观测当前月相** → 确定全球统一的农历日期（月相日期南北半球一致）
2. **北半球农历日期 + 15 = 南半球太极对立日**
3. **验证示例：** 北半球初八（上弦月，右半明亮）↔ 南半球廿三日（下弦月，左半明亮）——两者呈精确的镜像对称
4. **物理意义：** 朔望周期上相隔15天 = 180°对立，月相恰好反转，照亮一侧镜像对称

此验证方法无需任何仪器，仅需肉眼观测月球即可完成，为南半球历法体系提供了最直观的天文实证。

### 3.3 统一与对立

南北半球月相关系体现了太极哲学中"统一"与"对立"的辩证统一：

- **统一性：** 月相日期全球一致；满月处处圆满——此为"太极一气"
- **对立性：** 被照亮一侧镜像对称；盈亏方向相反——此为"阴阳分判"
- **太极义：** 朔望周期上180°对立 = 太极双鱼的旋转对称——北半球的"阳鱼"与南半球的"阴鱼"同处一图，互为镜像，相反而相成

## 四、南半球星象定节气

### 4.1 半人马座指针法

半人马座α星（南门二，视星等-0.27）与β星（马腹一，视星等0.61）构成的连线向南延伸约4.5倍，即指向南天极。此法功能类似于北半球的北斗七星指极法。

**定季节：**

- 指针正南 → 南半球冬季（6月前后）
- 指针正东 → 南半球春季（9月前后）
- 指针正北 → 南半球夏季（12月前后）
- 指针正西 → 南半球秋季（3月前后）

### 4.2 天蝎座心宿二法

心宿二（Antares，视星等1.0）是少数南北半球均易辨认的亮星，其季节指示意义在南北半球恰好相反：

- **北半球：** 心宿二黄昏高悬 = 盛夏
- **南半球：** 心宿二黄昏高悬 = 隆冬

同一颗星，同一时刻，南北半球的季节含义截然相反——这正是太极对称在天象上的直接体现。心宿二法精度最高，且南北半球共享同一参照星，是验证节气对称性的最佳方法。

### 4.3 老人星法与麦哲伦星云法

**老人星法：**

老人星（Canopus，视星等-0.74，全天第二亮星）的地平高度随季节变化：

- 高度角大 → 南半球冬季
- 高度角小 → 南半球夏季

**麦哲伦星云法：**

大小麦哲伦星云绕南天极旋转，其方位角指示季节：

- 星云位于正南 → 冬季
- 星云位于正东 → 春季
- 星云位于正北 → 夏季
- 星云位于正西 → 秋季

### 4.4 综合验证方案

| 优先级 | 方法 | 优势 | 适用范围 |
|:------:|------|------|----------|
| 1 | 心宿二法 | 精度最高，南北半球共享参照星 | 南北半球通用 |

## 五、科里奥利力与旋转方向

### 5.1 南北半球旋转对比表

| 系统类型 | 北半球 | 南半球 |
|---------|--------|--------|
| 台风/飓风/气旋（低压系统） | 逆时针 ↺ | 顺时针 ↻ |
| 反气旋（高压系统） | 顺时针 ↻ | 逆时针 ↺ |
| 大洋环流（副热带） | 顺时针 ↻ | 逆时针 ↺ |
| 地球自转方向（俯视极点） | 逆时针 | 顺时针 |

### 5.2 数学本质

科里奥利力的矢量表达式为：

$$\vec{F}_C = -2m(\vec{\omega} \times \vec{v})$$

其中 *m* 为物体质量，*ω* 为地球自转角速度矢量，*v* 为物体运动速度矢量。

- **北半球：** 科里奥利力使运动物体向右偏转 → 气旋逆时针旋转
- **南半球：** 科里奥利力使运动物体向左偏转 → 气旋顺时针旋转

**重要说明：** 科里奥利力仅影响大尺度系统（数百公里以上），小尺度现象（如浴缸排水方向）不受其显著影响。南北半球旋转方向的对称反转，是地球物理层面太极对称的直观体现——北半球的逆时针对应南半球的顺时针，恰如太极双鱼的旋转方向相反。

## 六、四柱八字与时辰

### 6.1 时辰与当地时间

时辰的本质是当地太阳时，而非统一的标准时间。同一时刻，不同时区的城市可能处于不同的时辰。例如，北京时间子时（23:00-01:00）对应悉尼时间凌晨2:00-4:00，已进入丑时。

因此，南半球历法的时辰计算必须基于当地太阳时，应用程序支持城市/时区选择，自动换算为正确的时辰。

### 6.2 南北半球四柱差异

四柱（年柱、月柱、日柱、时柱）在南北半球之间可能产生系统性差异：

| 四柱 | 差异来源 | 是否可能不同 |
|------|---------|:----------:|
| **年柱** | 年界在立春，南北半球立春相隔6个月 | 是 |
| **月柱** | 月界在节气，南北节气对称相反，地支不同 | 是 |
| **日柱** | 日界在子时，不同时区可能跨日 | 是 |
| **时柱** | 直接取决于当地时辰 | 必然不同 |

**年柱差异示例：** 北半球2026年2月4日立春后进入丙午年，而南半球此时为2025年8月，尚未到达南半球立春（对应北半球立秋，约8月7日前后），故南半球仍为乙巳年。

### 6.3 太极对称四柱

南半球四柱与北半球四柱呈太极对称关系：

$$\text{天干}_{南} = (\text{天干}_{北} + 6) \bmod 10$$

$$\text{地支}_{南} = (\text{地支}_{北} + 6) \bmod 12$$

此变换确保南北半球四柱在太极对称框架下严格对应。应用程序在配对城市间确保12小时时差，以实现精确的太极对称验证。
